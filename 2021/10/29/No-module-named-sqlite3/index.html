title: No module named ‘_sqlite3
date: 2021-10-29 16:00:00
tags: BUG

如标题提示，问题是因为我们安装的python版本缺少SQLite包，或者缺少.so文件。这个时候我们有两个解决办法。

<! –more–>
一：简单易懂，重装。

二：安装必要配置并重新编译python

  • 安装sqlite-devel或者libsqlite3-dev

  • 进入python文件夹，执行./configure --enable-loadable-sqlite-extensions && make && sudo make install

    如果出现权限问题，需要在执行make命令时，赋予root权限

按照以上两种任意一个办法运行完毕后，在CMD中执行sqlite3，进行命令行即表示成功，输入.quit可以退出命令行。
